# 42.humangl
HumanGL is a little school project that aim to introduce to hierarchical modeling and matrix stack manipulation.

## Technical requirements
It must use at least OpenGL 4.0

## Objectives
The project must display a character (like Minecraft) and correctly articulate the body parts. If the torso rotates, all the limbs must follow.
If a body part is resized, related parts must automatically reposition themselves.

The character model will have:
  - one head
  - one torso
  - two arms with
    - upper arm
    - forearm
  - two legs with
    - thigh
    - lower part

It should be able to walk, jump and stay put.

# Building from source
You can just download the source and run ```make``` at the root of the project. It will automatically download the dependencies and compile the project.
